# NEO-GPT: Adaptive AI Text Generation Interface

**Task 01 - Prodigy InfoTech Internship**

Neo-GPT is a professional-grade web application that leverages the **GPT-2 Large Language Model** to generate intelligent, long-form text. It features a modern Glassmorphic Bento UI, providing a high-end user experience with real-time performance metrics.

## ✨ Features

- **Adaptive Depth:** Choose between Short, Medium, and Long generation modes.
- **Smart Trimming:** Uses Regex logic to ensure the AI always finishes its last sentence.
- **Modern UI:** Responsive Bento-style grid with a Glassmorphism aesthetic.
- **Performance Metrics:** Real-time tracking of **Latency** and **Word Count**.
- **Safari Compatible:** Optimized with `-webkit` backdrop filters for iOS/macOS support.

---

## 🛠️ Technical Stack

- **AI Model:** GPT-2 (Hugging Face Transformers)
- **Backend:** Flask (Python 3.8+)
- **Frontend:** HTML5, CSS3 (Bento Grid), JavaScript (Fetch API)
- **Deployment:** Vercel / GitHub Actions
